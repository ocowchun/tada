## tada
> Terminal dashboard

## Requeirements
* Go

## Install
```
$ go get github.com/ocowchun/tada
$ cd ~/src/github.com/ocowchun/tada
$ git checkout develop
$ export TADA_GITHUB_TOKEN="your-personal-token"
$ go run tada run
```
